# Build Local & Desktop

## Описание
Единая цепочка для локальной сборки. Поддерживает как запуск веб-превью, так и сборку/подпись десктопных Tauri `.app` бинарников с разруливанием кэшей.

## Шаги
### Для Web проекторов (Dev / Preview)
1. **Запуск**: Выполни команду локальной сборки (`npm run dev` или аналогичную). Убедись, что порты поднялись.
2. **Превью**: Открой браузер.
// turbo
```bash
open http://localhost:PORT
```
3. Дождись подтверждения, что интерфейс видно.

### Для Desktop (Tauri / .app)
4. **Очистка**: Удали старые артефакты сборки и кэши (например `.next`, `cargo clean`).
5. **Сборка Frotend/Tauri**: Выполни долгую сборку, обернув ее в `caffeinate -i`, чтобы macOS не уснула. Например: `caffeinate -i pnpm run tauri build`.
6. **Подпись и перенос (macOS)**:
   - Если это релиз, перенеси `.app` в `/Applications` (избегание App Translocation).
   - Сними карантины: `xattr -cr /Applications/MyApp.app`.
7. Выведи отчет о готовности десктопного билда.
