# Quant Laboratory (End-to-End Quant Pipeline)

Интегрированный процесс поиска (скаутинга) алгоритмов и их последующего бэктеста.

### Этап 1: Скаутинг
// turbo
1. Запусти `search_web` для поиска свежих статей/проектов (2025-2026) по алгоритмическому трейдингу, MLX оптимизациям, статистическому арбитражу. Игнорируй простые MACD стратегии.
2. Пропарси содержимое лучших находок с помощью `read_url_content`.
3. Оцени применимость математической модели для ИИ-рефакторинга.
4. Создай `quant_research_report.md` с векторами атак/улучшений.

### Этап 2: Бэктест и Моделирование
5. Если пользователь выбирает конкретную стратегию, подгрузи исторические данные или подключи скрипты парсеров.
6. Напиши логику симуляции на Python (Pandas/MLX).
7. Запусти расчет Sharpe ratio, Max Drawdown, EV (Expected Value) и PnL.
8. Подготовь агрегированный репорт о прибыльности/рисках.
