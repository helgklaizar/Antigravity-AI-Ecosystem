# ИИ Клонировщик Сайтов (Website Cloner)

> Вдохновлено `JCodesMore/ai-website-cloner-template`.

## Суть
Когда нужно быстро перенести дизайн или структуру с референса в наш проект (Tailwind / CSS / React), этот воркфлоу координирует процесс извлечения DOM и стилей.

## Триггеры
- Пользователь скинул URL и сказал "сделай как тут".
- Нужно быстро собрать MVP лендинга на базе конкурента.

## Шаги

1. **Анализ Референса**:
   - Использовать браузерный субагент (если доступно) или curl/read_url_content для загрузки HTML-дерева страницы.
   - Извлечь основные цвета, шрифты и структуру (Hero секция, Features, Footer).

2. **Составление Скелета**:
   - Написать `implementation_plan.md` с маппингом: какие компоненты нам понадобятся создать в React/Vue/HTML.
   - Обновить `DESIGN.md` новыми токенами, если нужно.

3. **Генерация Кода**:
   - Создать базовые компоненты по плану.
   - Применить стили (Vanilla CSS или Tailwind, согласно стандартам проекта).

4. **QA и Полировка**:
   - Сравнить результат с оригиналом.
   - Вызвать `react-doctor` или `qa-ui-design-reviewer` для проверки на адаптивность и семантику.
