# Автономный Исследователь (Background Researcher)

> Вдохновлено `karpathy/autoresearch`.

## Суть
Этот воркфлоу запускает фонового агента для глубокого исследования (Deep Research). Агент работает автономно, читает документацию, собирает бенчмарки или анализирует конкурентов, после чего выдает готовый `research_report.md`.

## Триггеры
- Когда нужно "разобраться в новой библиотеке X".
- Когда нужно "собрать список лучших практик для Y".
- Когда нужно "проанализировать конкурентов Z".

## Шаги

1. **Инициализация**:
   - Агент получает вводную от пользователя (тема исследования).
   - Агент создает папку `research/<topic_name>/` для сбора артефактов.

2. **Сбор данных (Broad Search)**:
   - Использование `search_web` и `read_url_content` для поиска статей, GitHub репозиториев и документации.
   - Скачивание PDF или Markdown файлов.

3. **Синтез (Deep Analysis)**:
   - Анализ собранных данных. Выделение ключевых паттернов, плюсов и минусов.
   - Создание структуры отчета.

4. **Финальный отчет (Report Generation)**:
   - Формирование `research_report.md` в корне проекта или директории `research/`.
   - Отчет должен содержать: Executive Summary, Ключевые Находки, Ссылки на источники, Рекомендации к внедрению.
