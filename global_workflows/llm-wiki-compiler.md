# LLM Wiki Compiler
Этот воркфлоу отвечает за сборку и поддержание "Анти-RAG" базы знаний (паттерн LLM Wiki от Андрея Карпаты). 
Вместо векторного поиска агент регулярно индексирует новые исходники и обновляет файлы в формате Markdown.

## Step 1: Ingest (Сбор данных)
Анализирует указанную директорию или файлы и извлекает сырые знания.
// turbo
```bash
echo "Запуск анализатора исходников..."
# python scripts/llm_wiki_ingest.py --source ./docs --output ./wiki/raw
```

## Step 2: Compile (Генерация Вики)
Запускает LLM для обновления страниц сущностей, связей и маркдаун файлов на основе новых данных, устраняя противоречия.
// turbo
```bash
echo "Компиляция базы знаний..."
# python scripts/llm_wiki_compile.py --raw ./wiki/raw --dest ./wiki/pages
```

> [!TIP]
> LLM Wiki позволяет вашему агенту не терять контекст и не галлюцинировать: знания компайлятся один раз.
