# Autonomous Issue Triage

Автономный решатель конкретных тасок/багов. Позволяет делегировать мелкие и средние баги агенту "под ключ".

// turbo-all
1. Запроси у пользователя ссылку/номер issue (в GitHub или локальном TODO.md).
2. Скачай и проанализируй описание бага (через `read_url_content` или анализ локальных файлов).
3. **Локализация:** Используй `grep_search` и анализ файлов проекта для нахождения участка кода, ответственного за описываемое поведение.
4. Выяви Root Cause (коренную причину). Если необходимо протестировать поведение, напиши скрипт-пробник в директорию `scratch/` и выполни.
5. **Auto-Fix:** Примени `replace_file_content` или мульти-реплэйс для внедрения исправления.
6. Вызови `npm run test` (или эквивалент), чтобы удостовериться, что фикс сработал и старые тесты не упали.
7. Сгенерируй подробный diff-репорт об изменениях для ревью пользователем (Или предложи закоммитить).
