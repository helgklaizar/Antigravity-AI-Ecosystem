# 🍏 Mac Port Factory (/mac-port-factory)

**Описание:** Автоматизированный воркфлоу-генератор. Агент выступает в роли "Хантера технологий". Он самостоятельно ищет хайповый эксклюзив из мира NVIDIA/Linux, продумывает, как переписать его на Metal/MLX, и подготавливает всю инфраструктуру проекта под ключ внутри каталога `for-mac/`.

## 🛠 Контекст
- **Целевой агент:** `architect` или `developer`.
- **Локация:** Все генерации строго производятся в папке `/Users/klai/AI/for-mac/`.

## ⚙️ Этапы (Steps)

### 1. 🔍 Дискавери (Поиск цели)
// turbo
Используй инструмент `search_web` для поиска трендов 2026 года в мире AI, которые жестко привязаны к CUDA / Triton или требуют непомерных объемов VRAM. 
*Критерий:* У проекта не должно быть качественного нативного порта под Mac (MLX).
Выбери **один** самый сочный и перспективный проект.

### 2. 🧠 Архитектурный Штурм
Спроектируй концепцию портирования:
- Что будет делать наша мак-версия?
- За счет чего мы получим ускорение (Metal Shading Language, Zero-Copy память маков, кэширование ANE)?
- Как мы назовем проект (например: `mlx-[оригинал]`)?

### 3. 🏗 Инфраструктурный Деплой
// turbo-all
Без лишних вопросов к пользователю, разверни проект физически с помощью `add_to_workspace` или `write_to_file`:

1. Создай дирректорию в `~/AI/for-mac/[твой-проект]/`.
2. Создай файл `~/AI/for-mac/[твой-проект]/README.md`.
3. Оформи `README.md` по **строгим стандартам из воркфлоу `/github-publish-mac`**. 
   *Обязательные элементы внутри markdown:*
   - Центрированный заголовок с HTML тегами.
   - Бейджи (Apple Silicon, MLX/MPS, License).
   - Блок экосистемы: `> 🍏 **Part of the Mac AI Ecosystem Initiative**` (кстати, добавь сюда ссылки на другие наши проекты, например: TurboQuant, MLX Flamegraph).
   - Четкий список тасок (чекбоксов) в `🗺 Roadmap & Tasks`, расписанный на v0.1, v0.2.

### 4. 🚀 Готовность
Выведи пользователю в чат краткое сообщение: "Поймал крупную рыбу [Оригинал]. Создал клон `[Наш Порт]`. Папка готова, роадмап расписан. Идем кодить?".
