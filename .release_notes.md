# Что нового в релизе v2.2.0

В этом обновлении мы значительно улучшили статус-бар (AntigravityBar) и добавили мощные агентурные навыки для проектирования и планирования архитектуры.

## 📊 AntigravityBar (Статус-бар)
- **Системные Метрики**: Интегрирован мониторинг загрузки CPU, GPU и RAM в реальном времени.
- **Цветовая Индикация**: Добавлены цветовые индикаторы нагрузки (Зеленый, Желтый, Оранжевый, Красный) для быстрого визуального считывания состояния системы.
- **Редизайн Layout**: Переработан интерфейс в стиле профессионального Dashboard. Улучшена консистентность, добавлена строгая сетка, оптимизировано отображение кэша и добавлены новые иконки для приоритизации групп моделей (Gemini 3.1 Pro, Claude, O1 и др.).
- **Process Manager**: Добавлена новая логика управления процессами и сбором системной статистики (`SystemStats.swift`, `ProcessManager.swift`).

## 🧠 Новые Глобальные Workflows (AI Skills)
Интегрировано 5 новых мощных агентурных навыков для улучшения взаимодействия:
- **`/caveman`** — ультра-короткий режим общения для экономии токенов и быстрого получения сухой технической выжимки без "воды".
- **`/grill-me`** — режим безжалостного интервью для стресс-тестирования архитектуры и спецификаций перед реализацией.
- **`/ubiquitous-language`** — инструмент для автоматического извлечения словаря предметной области (DDD) из контекста беседы для предотвращения путаницы в терминах.
- **`/zoom-out`** — макрос для принудительного поднятия агента на уровень абстракции выше для обзора всей архитектуры в целом.
- **`/request-refactor-plan`** — формализованный процесс создания детального плана безопасного инкрементального рефакторинга с атомарными коммитами.
