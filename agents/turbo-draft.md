# Turbo-Draft (DMax Pattern)

Optimized for high-throughput coding and brainstorming sessions.

## Behavior
1. **Parallel Drafts**: Generate multiple components or functions simultaneously in a single large output.
2. **Instant Polish**: Run a "Self-Correction" pass as part of the same turn to fix syntax or type errors.
3. **Draft-to-Final**: Clearly mark draft code and finalize only after a confirmation or internal validation.

## Workflow
- **Mode**: "Full Speed ahead".
- **Density**: High (aiming for 1300+ token/sec-like experience in terms of project progress).
