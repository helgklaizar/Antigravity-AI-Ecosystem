# 🧹 Registry Cleaner (Очистка системы)

**Описание:** Этот воркфлоу анализирует активность использования различных инструментов (скиллов, агентов и воркфлоу) в истории переписки и переносит те, что не использовались, в архивную папку, чтобы не захламлять основной контекст.

## ⚙️ Алгоритм работы

### 1. 🔍 Сбор инвентаря
// turbo
Необходимо составить полный список текущих файлов:
- Все воркфлоу из `~/.ai-ecosystem/workflows/`
- Все скиллы из `~/.ai-ecosystem/skills/`

### 2. 🧠 Проверка активности (Grep logs)
// turbo
Для каждого найденного инструмента нужно запустить поиск по логам памяти (`~/.ai-ecosystem/brain/`).
- Ищем упоминания формата `[Агент: <name>]`, `[Скилл: <name>]` или `[Воркфлоу: /<name>]`.
- Также проверяем наличие названия в заголовках недавних сессий.

### 3. 📋 Формирование отчета "Призраков"
Выведи список инструментов, по которым **не найдено ни одного упоминания** в недавней истории (например, в последних 10 сессиях).

### 4. 📦 Архивация
// turbo
Если пользователь подтверждает, выполни следующие действия:
1. Создай директории `_archive` внутри `global_workflows` и `knowledge/skills`.
2. Перемести (mv) неиспользуемые файлы в соответствующие папки архива.
3. Обнови статус системы.

## 🔄 Системный контекст
При запуске выведи:
`🔄 **Контекст:** [Агент: devops] | [Скилл: None] | [Воркфлоу: /registry-cleaner]`
