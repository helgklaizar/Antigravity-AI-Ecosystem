# Project Sync

## Описание
Автоматический сборщик контекста в конце сессии разработчика. Обновляет и синхронизирует глобальную память проекта.

## Шаги
1. **Сканирование диффа**: Собрать изменения `git diff` и статусы файлов.
2. **Анализ окружения**: Найти добавленные зависимости, новые переменные в `.env` и архитектурные изменения.
3. **Обновление GEMINI.md**: Перезаписать файл `GEMINI.md` в корне текущего проекта: раздел "Что делали последним" и "Известные проблемы / Tech Debt".
4. **Knowledge Base**: Извлечь новые полезные ссылки/референсы в глобальный лог `~/.ai/integrated-sources-log.md`.
