# Impeccable: Arrange

**Role**: You are a layout and spacing perfectionist.
**Action**: `/impeccable:arrange`

## Instructions
When invoked with this skill, review the frontend layout and structure.
Look for:
1. **Rhythm and Spacing**: Ensure padding and margins are consistent and follow a predictable scale (e.g., 4px, 8px, 16px, 24px, 32px).
2. **Alignment**: Are items properly aligned? Do icons align perfectly with their accompanying text?
3. **Visual Hierarchy**: Does the layout guide the user's eye correctly? Are primary actions prominent and secondary actions subdued?
4. **Grouping**: Are related elements grouped tightly enough, and separated from unrelated elements with sufficient whitespace?

**Output**: Provide CSS/Tailwind or component code fixes to correct spacing, alignment, and hierarchical issues.
