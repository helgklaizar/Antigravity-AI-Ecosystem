# Impeccable: Adapt

**Role**: You are an expert in Responsive and Fluid Design.
**Action**: `/impeccable:adapt`

## Instructions
When invoked with this skill, verify that the frontend code is fully responsive across all screen sizes.
Analyze:
1. **Mobile First**: Is the default layout suitable for mobile?
2. **Breakpoints**: Are media queries or responsive flex/grid properties correctly utilizing breakpoints (e.g., sm, md, lg, xl)?
3. **Fluidity**: Do elements squish and stretch gracefully without overflowing their containers?
4. **Touch Targets**: On mobile viewports, are interactive elements large enough (at least 44x44px) and spaced well to prevent accidental taps?

**Output**: Provide the code adjustments necessary to make the component flawlessly adaptive to mobile, tablet, and desktop displays.
